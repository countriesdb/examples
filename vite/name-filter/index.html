Name Filter - Vite

This example demonstrates custom name filtering using JavaScript callback functions. Name filters override all other name sources (romanization, local variant, language preferences), giving you complete control over displayed names.

Country Filter: Filters out CA and GB, customizes US and EG names.
Subdivision Filter: Filters out EG-GZ, US-CA, US-NY, and customizes EG-C and EG-ALX names.
