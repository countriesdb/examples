Subdivision Romanization Preference - Vite

This example prefers UN romanization, then BGN, for official subdivisions by setting subdivisionRomanizationPreference: 'un,bgn' in window.CountriesDBConfig. Note: This only works with official ISO subdivisions, so we use data-prefer-official on the subdivision select. This option is ignored when extended subdivisions are active.
